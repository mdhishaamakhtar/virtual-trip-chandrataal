# Chandra Taal

An immersive editorial storytelling site that recreates a solo mountain trip from Lucknow to Chandra Taal lake in Spiti Valley, told across six chapters. Each chapter is a full-screen, cinematic experience — one paragraph at a time. The site is a gift, made for one reader, meant to feel like opening something handmade rather than loading a webpage.

## Running locally

It's a single static page. Open `index.html` directly in a browser, or serve the directory:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Structure

- `index.html` — the entire site (markup, styles, and behavior)
- `assets/` — imagery and audio used across the chapters
- `PRODUCT.md` — product brief: voice, principles, and constraints
